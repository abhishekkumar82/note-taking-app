# 📝 Note Taking App (MERN + Razorpay)

A full-stack Note Taking Application built using the MERN stack with premium features and Razorpay payment integration.

---

## 🚀 Features

* 📝 Create, edit, delete notes
* 📂 Folder-based organization
* 🔒 Locked notes with PIN
* 📅 Personal Diary (Premium)
* 📊 Habit Tracker (Premium)
* ⏰ Smart reminders with notifications
* 💳 Razorpay payment integration
* 💎 Premium subscription system

---

## 🛠 Tech Stack

### Frontend

* React.js
* Tailwind CSS / Custom CSS
* Axios

### Backend

* Node.js
* Express.js
* MongoDB (Mongoose)

### Other

* Razorpay Payment Gateway
* JWT Authentication
* Google OAuth (optional)

---

## 📂 Project Structure

```
client/    → Frontend (React)
server/    → Backend (Node + Express)
```

---

## ⚙️ Installation

### 1. Clone repo

```
git clone https://github.com/YOUR_USERNAME/note-taking-app.git
cd note-taking-app
```

### 2. Install dependencies

```
cd client && npm install
cd ../server && npm install
```

---

## 🔑 Environment Variables

Create `.env` file in **server/**

```
PORT=9090
MONGO_URI=your_mongodb_url
JWT_SECRET=your_secret
RAZORPAY_KEY_ID=your_key
RAZORPAY_KEY_SECRET=your_secret
```

---

## ▶️ Run Project

```
npm run dev
```
